# Portal QIE Lab

一个用于归档和发布独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为静态页面集合的存储与展示空间。

## 项目简介

Portal QIE Lab 旨在收集、整理并提供一系列自包含的 HTML 页面。这些页面可能涉及各种主题或功能，但彼此独立，不依赖特定后端服务。  
本仓库适合需要快速部署或分享单个页面文件的场景。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放所有归档的 HTML 页面
│   ├── example-1.html  # 示例页面
│   └── example-2.html
├── assets/             # 页面共用的资源文件（如图片、样式、脚本）
│   ├── css/
│   └── js/
└── README.md           # 本文件
```

- **pages/**：主要存放所有独立 HTML 页面，每个页面均为完整可运行的 HTML 文档。
- **assets/**：存放页面间共享的静态资源，便于统一管理和维护。

## 页面归档说明

- 每个页面应保持独立，避免引用外部不可控资源（除非必要）。
- 页面命名建议使用英文小写，单词间以连字符分隔，例如 `about-page.html`。
- 如有特殊说明，可在页面内注释或在本仓库的 `docs/` 目录添加文档。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时请遵循现有目录结构。
- 若需贡献页面，请提交 Pull Request，并确保页面内容不违反相关法律法规。
- 仓库不面向特定用户群体，不承诺任何形式的更新频率或技术支持。
